# Enterprise Network Simulation (Cisco Packet Tracer)

## 📌 Project Overview
This project simulates a small enterprise network using VLAN segmentation and inter-VLAN routing.

## 🧠 Key Concepts
- VLAN Configuration
- Inter-VLAN Routing (Router-on-a-Stick)
- IP Addressing Design
- Network Testing & Troubleshooting

## 🛠 Tools Used
- Cisco Packet Tracer

## 🌐 Network Design
- 1 Router (R1)
- 1 Core Switch (SW1)
- Multiple VLANs (10, 20, 30)

## 📂 Project Structure
- configs/ → Router & Switch configurations
- docs/ → IP plan, VLAN design, test results
- topology/ → Network design explanation
- simulation/ → Packet Tracer simulation
