# Network Topology

## Overview

This project simulates a small enterprise network using:

- 1 Router (R1)
- 1 Core Switch (SW1)
- 2 Access Switches (optional extension)
- Multiple PCs

## Design

- Router connected to switch via trunk link
- VLAN segmentation implemented
- PCs connected to access ports

## Diagram Description

Internet → Router → Core Switch → Access Switches → PCs

## Tools Used

Cisco Packet Tracer
