# Simulation File

## Tool Used

Cisco Packet Tracer

## File

The simulation file (.pkt) demonstrates:

- VLAN configuration
- Inter-VLAN routing
- Network connectivity tests

## How to Use

1. Open file in Cisco Packet Tracer
2. Switch to Simulation Mode
3. Run ping tests
4. Observe packet flow

## Status

Working simulation with successful connectivity across VLANs
